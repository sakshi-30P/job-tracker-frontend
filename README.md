# Job Tracker System - Frontend

**Technologies:** React, HTML, CSS, Bootstrap, Axios

---

## **Project Overview**
React frontend for Job Tracker System.  
Allows users to view, add, update, and delete jobs with a clean UI. Connects to Spring Boot backend via REST APIs.

---

## **Features**
- Job listing with search and filter  
- User login/register  
- Add/Edit/Delete jobs  
- Responsive design using Bootstrap  

---

## **How to Run**
1. Clone the repo:
```bash
git clone https://github.com/sakshi-30P/job-tracker-frontend.git

2. Install dependencies:
npm install

3.Start frontend:
npm start

Frontend runs on: http://localhost:3000
